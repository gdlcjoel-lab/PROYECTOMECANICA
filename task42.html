Section 4.2 — Sensitivity Analysis

Effect of Coriolis Force on Trajectory

What happens if we ignore Earth's rotation? We compare the trajectory with ω = 7.27×10⁻⁵ rad/s against ω = 0, isolating the Coriolis pseudo-force and quantifying its effect on the landing point.

ω = 7.27×10⁻⁵ rad/s vs ω = 0 Launch at equator ISA · T = 1500 N
⚙️
Physical Setup

Case B — without Coriolis
ω = 0 (disabled)
Coriolis acceleration
a⃗_C = −2 ω⃗ × v⃗
At equator (ω⃗ = ω ŷ)
a_Cx = −2ω·vz (East-West)
Thrust / atmosphere

Physical meaning: a rocket moving upward (v_z > 0) experiences a Coriolis force pointing westward (−x). On descent (v_z < 0) it deflects eastward, partially cancelling. Net result: a small westward shift at the landing site. The North-South component is zero at the equator.
🔬
Analysis Questions
What the simulation answers
Q1 — How does the trajectory change?
Coriolis deflects the rocket westward during ascent and eastward during descent. The net landing displacement and horizontal range are compared between both cases.
Q2 — Which direction (East-West or North-South) is most affected?
At the equator: a_Cx = −2ω·v_z (East-West, non-zero), a_Cy = 0 (North-South, zero). The East-West direction dominates completely.

"""
Task 4.2 — Remove Coriolis Force
Compare omega = OMEGA_EARTH (base) vs omega = 0
"""
import numpy as np
from rocket_simulation import simulate_rocket, compute_metrics, OMEGA_EARTH
from plot_utils import plot_comparison

OMEGA_CASES = [OMEGA_EARTH, 0.0]
LABELS      = ['Coriolis ON (base)', 'Coriolis OFF (omega=0)']
COLORS      = ['#0D2B55', '#B07D2A']

results, metrics_list = [], []
for omega, label in zip(OMEGA_CASES, LABELS):
    t, r, v, drag, mass, accel = simulate_rocket(omega=omega)
    m_obj = compute_metrics(t, r, v)
    results.append((t, r, v, drag, mass, accel))
    metrics_list.append(m_obj)

m_on, m_off = metrics_list
print(f"Coriolis ON:  x={m_on['x_land']:+.4f}m  y={m_on['y_land']:+.2e}m")
print(f"Coriolis OFF: x={m_off['x_land']:+.4f}m  y={m_off['y_land']:+.2e}m")
print(f"Net west deflection: {abs(m_on['x_land']):.4f} m")

# Interactive 4-panel plot (altitude, East, North, horizontal projection)
plot_comparison(panels=[...], labels=LABELS, colors=COLORS,
                suptitle='Task 4.2 — Effect of Coriolis Force')
