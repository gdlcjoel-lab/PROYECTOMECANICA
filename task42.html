Section 4.2 — Sensitivity Analysis

Effect of the Coriolis Force

Remove Earth's rotation from the model and compare the trajectory against the base case. Quantify the displacement introduced by the Coriolis pseudo-force and identify which spatial direction is most affected.

ω = 7.27×10⁻⁵ rad/s vs ω = 0 Launch at equator T = 1500 N · ISA
📋
Exercise Statement
Section 4.2 of the project brief

Disable the Coriolis effect by setting ω = 0 and compare the resulting trajectory with the base case (ω = 7.27×10⁻⁵ rad/s). All other parameters remain identical to the base case.

Q1How does the trajectory change when the Coriolis force is removed? Compare the landing coordinates (x, y) for both cases.
Q2Which spatial direction — East-West or North-South — is most affected by the Coriolis force? Justify physically using the expression a⃗_C = −2ω⃗ × v⃗ at the equator.
⚙️
Physical Setup
Only Earth angular velocity ω varies
Case A — Coriolis ON
ω = 7.27×10⁻⁵ rad/s
Case B — Coriolis OFF
ω = 0 (disabled)
Coriolis acceleration
a⃗_C = −2 ω⃗ × v⃗
At equator (ω⃗ = ω ŷ)
a_Cx = −2ω·vz · a_Cy = 0
Fixed thrust / atmosphere
T = 1500 N · ISA · 90 kg prop
Frame
Local inertial (x=East, y=North, z=Up)
Physical mechanism: a rising rocket (v_z > 0) experiences a_Cx = −2ω·v_z < 0 — deflected westward. On descent (v_z < 0) it deflects eastward, partially cancelling. Net result: a small but measurable westward landing displacement. North-South component is exactly zero at the equator (a_Cy = 0 when ω⃗ ∥ ŷ).

Analysis
Answers derived from simulation
Q1 — Trajectory change
With Coriolis ON the rocket lands 3.044 m west of the launch site (−x direction). Without Coriolis it lands exactly at the origin. Maximum altitude and flight time are virtually identical in both cases — the Coriolis force affects the horizontal trajectory but not the vertical dynamics significantly.
Q2 — Direction most affected
East-West (x) is completely dominant. At the equator: a_Cx = −2ω·vz (depends on vertical velocity, non-zero throughout flight) while a_Cy = 0 exactly (the North component of ω⃗ × v⃗ is zero when ω⃗ ∥ ŷ and the rocket moves in the x-z plane). North deviation is identically zero by symmetry.

"""
Task 4.2 — Remove Coriolis Force
Compare omega = OMEGA_EARTH (base) vs omega = 0
"""
import numpy as np
from rocket_simulation import simulate_rocket, compute_metrics, print_metrics, OMEGA_EARTH
from plot_utils import plot_comparison

OMEGA_CASES = [OMEGA_EARTH, 0.0]
LABELS      = ['Coriolis ON (base)', 'Coriolis OFF (omega=0)']
COLORS      = ['#0D2B55', '#B07D2A']
LINESTYLES  = ['--', '-']

results, metrics_list = [], []
for omega, label in zip(OMEGA_CASES, LABELS):
    t, r, v, drag, mass, accel = simulate_rocket(omega=omega)
    m_obj = compute_metrics(t, r, v)
    results.append((t, r, v, drag, mass, accel))
    metrics_list.append(m_obj)
    print_metrics(m_obj, label)

m_on, m_off = metrics_list
print(f"Coriolis ON:  x={m_on['x_land']:+.4f}m  y={m_on['y_land']:+.2e}m")
print(f"Coriolis OFF: x={m_off['x_land']:+.4f}m  y={m_off['y_land']:+.2e}m")
print(f"Net west deflection: {abs(m_on['x_land']):.4f} m")
# Conclusion: East-West most affected. North-South exactly zero at equator.
