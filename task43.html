Section 4.3 — Sensitivity Analysis

Atmosphere Model Comparison

How much does the atmospheric density model matter? We compare the ISA multi-layer model against a simplified single-exponential approximation ρ(z) = ρ₀·exp(−z/H), analysing differences in altitude, drag and speed.

ISA vs Exponential ρ(z) H = 8500 m · ρ₀ = 1.225 kg/m³ T = 1500 N · 90 kg propellant

Key difference: ISA has a density kink at the tropopause (11 km) — the lapse rate changes from −6.5 K/km to 0 K/km, causing a slower density drop. The exponential model decreases monotonically everywhere, underestimating density above ~7 km and overestimating it below ~5 km.
🔬
Analysis Questions
What the simulation answers
Q1 — How do maximum altitudes compare?
The exponential model predicts lower density above ~7 km, so the rocket experiences less drag and reaches a slightly higher apogee compared to ISA.
Q2 — How does drag evolve differently for each model?
ISA produces higher drag in the 7–15 km range (denser stratosphere). The exponential model decays smoothly with no kink. This difference is visible in the drag vs time plot.
Q3 — Which model is more physically realistic?
ISA — it captures the temperature inversion at the tropopause and has been validated against measured atmospheric profiles to <1% accuracy up to 85 km. The exponential is a useful analytical approximation only.

"""
Task 4.3 — Atmosphere Model Comparison
ISA multi-layer vs single-exponential rho(z) = rho0 * exp(-z/H)
"""
import numpy as np
from rocket_simulation import (simulate_rocket, compute_metrics,
    density_isa, density_exponential, RHO0, H_SCALE)
from plot_utils import plot_comparison

DENSITY_CASES = [density_isa, density_exponential]
LABELS        = ['ISA (multi-layer)', 'Exponential ρ₀·exp(−z/H)']
COLORS        = ['#0D2B55', '#B07D2A']

results, metrics_list = [], []
for density_func, label in zip(DENSITY_CASES, LABELS):
    t, r, v, drag, mass, accel = simulate_rocket(density_func=density_func)
    results.append((t, r, v, drag, mass, accel))
    metrics_list.append(compute_metrics(t, r, v))

m_isa, m_exp = metrics_list
print(f"ISA   z_max = {m_isa['z_max']/1000:.3f} km")
print(f"Exp   z_max = {m_exp['z_max']/1000:.3f} km")
print(f"Delta      = {(m_exp['z_max']-m_isa['z_max'])/1000:+.3f} km")

# Interactive plot: altitude, drag, speed, air density experienced
plot_comparison(panels=[...], labels=LABELS, colors=COLORS,
                suptitle='Task 4.3 — ISA vs Exponential Atmosphere')
