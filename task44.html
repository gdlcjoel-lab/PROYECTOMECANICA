Section 4.4 — Sensitivity Analysis

Effect of Propellant Mass on Trajectory

How does the amount of propellant loaded affect acceleration, burnout altitude and maximum apogee? We compare 10, 50 and 90 kg of propellant with fixed thrust and payload mass.

m_prop = 10 / 50 / 90 kg T = 1500 N fixed Payload MP = 10 kg ISA · Coriolis ON

Trade-off: less propellant → lighter rocket → higher initial acceleration (a₀ = T/m₀ − g₀). But more propellant → longer burn → higher burnout altitude and more delta-v. After burnout the rocket coasts ballistically — the higher the burnout altitude and velocity, the higher the apogee.
🔬
Analysis Questions
What the simulation answers
Q1 — How does the acceleration profile change with propellant mass?
Net initial acceleration a₀ = T/m₀ − g₀. A lighter rocket (10 kg prop) has higher a₀ but runs out of fuel quickly. The acceleration profile shape is also different because mass decreases faster proportionally for lighter rockets.
Q2 — How does burnout altitude change?
More propellant → longer burn → higher altitude at burnout, despite the rocket being heavier. The dotted vertical lines on the plot mark the burnout instant for each case.

"""
Task 4.4 — Propellant Mass Variation
10 kg -> 50 kg -> 90 kg, fixed T = 1500 N
"""
import numpy as np
from rocket_simulation import simulate_rocket, compute_metrics, T0, ISP, MP, G0
from plot_utils import plot_comparison

PROP_MASSES = [10.0, 50.0, 90.0]
LABELS      = ['m_prop = 10 kg', 'm_prop = 50 kg', 'm_prop = 90 kg']
MDOT = T0 / (G0 * ISP)  # 1.530 kg/s — same for all cases

results, metrics_list = [], []
for mp in PROP_MASSES:
    m0 = MP + mp; tb = mp / MDOT; a0 = T0/m0 - G0
    print(f"m_prop={mp:.0f}kg  m0={m0:.0f}kg  t_burn={tb:.2f}s  a0={a0:.2f}m/s²")
    t, r, v, drag, mass, accel = simulate_rocket(m_propellant=mp)
    results.append((t, r, v, drag, mass, accel))
    metrics_list.append(compute_metrics(t, r, v))

# Burnout altitudes
for mp, res in zip(PROP_MASSES, results):
    t_arr, r_arr = res[0], res[1]; tb = mp/MDOT
    idx = int(np.argmin(np.abs(t_arr - tb)))
    print(f"m_prop={mp:.0f}kg  z_burn={r_arr[idx,2]:.1f}m  z_max={metrics_list[...]['z_max']/1000:.2f}km")

# Interactive 4-panel plot (altitude, speed, mass, acceleration)
plot_comparison(panels=[...], labels=LABELS,
                suptitle='Task 4.4 — Propellant Mass Variation')
