Section 4.1 — Sensitivity Analysis

Effect of Thrust on Trajectory

Repeat the base-case simulation varying engine thrust across three levels, keeping all other parameters fixed. Analyse how altitude, burnout time and aerodynamic losses respond to the change.

T = 1000 / 1500 / 3000 N ISA · Coriolis ON m_prop = 90 kg · Δt = 0.1 s
📋
Exercise Statement
Section 4.1 of the project brief

Repeat the base-case simulation varying the engine thrust: T = 1000 N, 1500 N and 3000 N, keeping all other parameters constant (same propellant mass of 90 kg, same Isp, Cd, atmosphere model and Coriolis effect).

Q1How does the maximum altitude change with thrust? Explain the non-monotonic behaviour if observed.
Q2Does the rocket escape the dense atmosphere (troposphere boundary at 11 km) for each thrust value?
Q3How does the burnout time t_burn vary with thrust? Derive the analytical relationship.
⚙️

Key insight: all three cases carry identical propellant mass, so total impulse I = Isp·g₀·m_prop is the same. Yet T = 3000 N reaches less altitude than T = 1500 N — higher thrust means faster ascent through dense air, increasing drag losses (F_D ∝ v²). There is an optimal thrust between gravity-loss and drag-loss regimes.

Analysis
Answers derived from simulation
Q1 — Non-monotonic altitude response
T=1000 N (10.69 km) < T=1500 N (11.28 km) > T=3000 N (8.57 km). Same total impulse, but T=3000 N reaches burnout in only 29.4 s — still deep in the troposphere where ρ is high — generating much larger drag losses (F_D ∝ v²). T=1000 N has less drag but wastes more time fighting gravity at low velocity. T=1500 N is near-optimal.
Q2 — Troposphere escape (11 km boundary)
Only T = 1500 N crosses 11 km (barely, at 11.28 km). T = 1000 N reaches 10.69 km and T = 3000 N only 8.57 km — both remain fully within the troposphere. Clearing the tropopause significantly reduces drag on the ballistic coast phase.
Q3 — Burnout time relationship
ṁ = T / (g₀·Isp) → t_burn = m_prop / ṁ = m_prop·g₀·Isp / T. Inversely proportional to thrust: doubling T halves t_burn. T=1000N: 88.3 s · T=1500N: 58.8 s · T=3000N: 29.4 s.

"""
Task 4.1 — Thrust Variation
Compare T = 1000 N, 1500 N (base), 3000 N
"""
import numpy as np
from rocket_simulation import simulate_rocket, compute_metrics, print_metrics, G0, ISP
from plot_utils import plot_comparison

THRUST_CASES = [1000.0, 1500.0, 3000.0]
LABELS       = ['T = 1000 N', 'T = 1500 N (base)', 'T = 3000 N']
COLORS       = ['#5B9BD5', '#0D2B55', '#C0392B']
LINESTYLES   = ['-', '--', ':']
PROP_MASS    = 90.0   # kg — identical for all cases

results, metrics_list = [], []
for T in THRUST_CASES:
    mdot   = T / (G0 * ISP)
    t_burn = PROP_MASS / mdot
    print(f"T={T:.0f}N  mdot={mdot:.4f}kg/s  t_burn={t_burn:.2f}s")
    t, r, v, drag, mass, accel = simulate_rocket(thrust=T, m_propellant=PROP_MASS)
    results.append((t, r, v, drag, mass, accel))
    metrics_list.append(compute_metrics(t, r, v))
    print_metrics(metrics_list[-1], f"T = {T:.0f} N")

# Q1 — max altitude (non-monotonic)
for T, m in zip(THRUST_CASES, metrics_list):
    print(f"T={T:.0f}N  z_max={m['z_max']/1000:.3f}km")

# Q2 — troposphere escape
for T, m in zip(THRUST_CASES, metrics_list):
    print(f"T={T:.0f}N  escape={'YES' if m['z_max']>11000 else 'NO'}")

# Q3 — burnout times
for T in THRUST_CASES:
    mdot = T/(G0*ISP); print(f"T={T:.0f}N  t_burn={PROP_MASS/mdot:.2f}s")

# Interactive 4-panel plot with time slider
plot_comparison(
    panels=[...],   # altitude, speed, drag, mass
    labels=LABELS, colors=COLORS, linestyles=LINESTYLES,
    suptitle='Task 4.1 — Effect of Thrust on Rocket Trajectory',
)
