Section 4.1 — Sensitivity Analysis

Effect of Thrust on Trajectory

How does engine thrust change maximum altitude, burnout time, and drag losses? Three thrust levels compared with identical propellant mass — showing the trade-off between gravity losses and aerodynamic losses.

Key insight: all three cases carry identical propellant (same total impulse I = Isp·g₀·m_prop). Yet they reach different altitudes — the balance between gravity losses (low T, slow ascent) and drag losses (high T, high speed in dense air) determines the optimum thrust.
🔬
Analysis Questions
What the simulation answers
Q1 — How does maximum altitude change with thrust?
Higher T → shorter burn, more speed at low altitude → more drag loss in dense air. Lower T → longer burn, more time fighting gravity → more gravity loss. There is an optimal thrust between these extremes.
Q2 — Does the rocket escape the troposphere (> 11 km)?
The ISA tropopause at 11 km marks a sharp density reduction. Clearing it significantly reduces drag on the ballistic coast phase after burnout.
Q3 — How does burnout time scale with thrust?
t_burn = m_prop / ṁ = m_prop · g₀ · Isp / T — inversely proportional to thrust. Doubling T halves the burn time.

"""
Task 4.1 — Thrust Variation
Compare T = 1000 N, 1500 N (base), 3000 N
"""
import numpy as np
from rocket_simulation import simulate_rocket, compute_metrics, print_metrics, G0, ISP
from plot_utils import plot_comparison

THRUST_CASES = [1000.0, 1500.0, 3000.0]
LABELS       = ['T = 1000 N', 'T = 1500 N (base)', 'T = 3000 N']
COLORS       = ['#5B9BD5', '#0D2B55', '#C0392B']
LINESTYLES   = ['-', '--', ':']
PROP_MASS    = 90.0

results, metrics_list = [], []
for T in THRUST_CASES:
    mdot   = T / (G0 * ISP)
    t_burn = PROP_MASS / mdot
    t, r, v, drag, mass, accel = simulate_rocket(thrust=T, m_propellant=PROP_MASS)
    results.append((t, r, v, drag, mass, accel))
    metrics_list.append(compute_metrics(t, r, v))
    print_metrics(metrics_list[-1], f"T = {T:.0f} N  (t_burn = {t_burn:.2f} s)")

# Answers
for T, m in zip(THRUST_CASES, metrics_list):
    esc = 'YES' if m['z_max'] > 11000 else 'NO'
    print(f"T={T:.0f}N  z_max={m['z_max']/1000:.3f}km  troposphere_escape={esc}")

# Interactive 4-panel plot with time slider
plot_comparison(panels=[...], labels=LABELS, colors=COLORS,
                suptitle='Task 4.1 — Effect of Thrust on Rocket Trajectory')
